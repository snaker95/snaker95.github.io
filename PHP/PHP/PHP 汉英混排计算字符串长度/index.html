PHP 汉英混排计算字符串长度

在PHP后台做数据验证是必须的, 而限制字符串长度是常见不过的, 但是由于汉字在UTF-8编码中,长度却是3个, 所以,在汉英混排时计算字符串长度就不是很好确定了, 但是PHP中已提供了相关函数

一. 函数介绍

strlen() 和 mb_srelen()

二. 计算字符串占位

中英混排占位 (一个中文字符的占位是2，英文字符是1)

三. 数据库中存储长度

varchar、char 中文字符占位为1
