一. 基本思路

1. 把img标签中的src属性删掉, 此时, 添加属性 data_src, 把图片链接放在该属性下

2. 通过使用img标签添加背景图片, 充当懒加载前的显示图片

3. 当该img标签在浏览器显示区内时, 通过js实现把img标签 中的data_src的链接赋值到img标签中的src属性
