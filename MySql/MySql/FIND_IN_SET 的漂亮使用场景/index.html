FIND_IN_SET 的漂亮使用场景

　　在数据库中,可能会出现一对多的情况, 有时候数据库就会设计这么一个字段来记录该情况, 例如: 该字段(father)记录自己的所属分类:１，２，３４，６７,说明当前自己属于１／２／３４／６７的分类；
　　在一定的场景下，　需要查询某一分类下的所有子级，　例如：查询所有67分类下的所有子类, 这样的数据库看起来好像很蛋疼了…. 但是不要怕,今天一不小心发现了一个MySQL的高级函数(FIND_IN_SET) , 类是与 IN,

重点
