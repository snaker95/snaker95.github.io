MySql主从数据库

随着数据访问量逐渐增大,数据库的承受力也达到了最大值,为了提交效率,数据库使用主从复制、读写分离的技术进行解决.
怎么安装MySql数据库,在这里就不多说了,只说主从赋值:

1. 主从服务器分别做以下操作

1.1 版本一致
1.2 初始化表,并在后台启动MySql
1.3 修改root的密码

2. 修改主服务器master

3. 修改从服务器slave

4. 重启2台服务器的MySql

5. 在主服务器上建立账户并授权slave

6. 登录主服务器的mysql，查询master的状态
